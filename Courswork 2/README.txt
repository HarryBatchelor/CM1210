To run part 1 of the coursework, first check there is a file called GPT2.txt and stopwords.txt. Then open the command line and navigate to the files. In the command line type "javac Stopwords.java" then "java Stopwords" and it should run

For part 2 make sure you once again have a file called GPT2.txt and stopwords.txt. Then open the command line and navigate to the files. Then type in "javac Sorting.java" then "java Sorting" and it will run

For part 3 Open the command line and navigate to the files. Then type in "javac MyArrayQueue.java" then "java MyArrayQueue" and it should run
